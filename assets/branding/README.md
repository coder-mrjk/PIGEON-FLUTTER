# Branding assets

Put your images here. Suggested sizes and purposes:

- logo/
  - pigeon_logo.svg (vector recommended)
  - app_icon_512.png (512x512 PNG for app shortcuts)
- backgrounds/
  - app_bg_light.png (large background for light theme)
  - app_bg_dark.png (large background for dark theme)
- ai/
  - ai_logo.svg (logo shown in AI popups/headers)

Web favicon and title:
- Update web/favicon.png (32x32 or 64x64) to change the Chrome tab icon.
- The tab title is set in web/index.html <title>.
